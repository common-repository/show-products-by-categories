Plugin for WordPress WooCommerce. For every page you can tune what product must show, while creation page or post you select in existing caregories what must be show.

== Description ==

The plugin adds a unit with a list of all existing product categories, page creation and editing of pages and fasting. As well as adding a new widget that you want to add to any existing location for widgets. After selecting the desired category, and add widgets, the page will appear in a random order, only items that are included in the selected categories.
